# Install RVM + Ruby on Rails on Mac Os X

Plan for install:

 * Brew
 * RVM
 * Ruby on Rails


### Brew

```console
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
### RVM

```console
sudo \curl -sSL https://get.rvm.io | bash -s stable --rails --autolibs=enable
```

### Ruby on Rails
```console
gem install rails
```
